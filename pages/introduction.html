GappsLite is a dedicated google app for the android operating system

Initially I was looking for lightweight custom Google apps, because I only used Google Apps to download games and applications from Playstore. i found a lightweight custom gapps but it's not quite what i want yet

It was then that I created my own custom gapps which are lightweight, small and support multiple devices that can be installed system or Systemless

After I made GappsLite on the 15th, I thought to make GappsLite installable on various devices and versions of Android with one flashable package. that's where "GappsLite++" was born with a separate package from the usual GappsLite.

in 08-2020 i have made "GappsLite Controller" for extra features of "GappsLite" or "GappsLite++" with some tweaks used by users

after several updates GappsLite added variants to complement user needs, with "GappsLite variant" you can choose which package you need.

In May 2022, we also added "addon/packages" so users can install which packages they need manually.

in july 2022, we made "GappsLite Remover" as a solution for roms that include gapps and remove them, then you can install GappsLite

so that's an explanation and some introduction you can read further below to find out "project GappsLite".

GappsLite++

GappsLite++ is a development of regular GappsLite, with GappsLite++ you can use it for cross-architectures and android versions in 1 flashable ZIP, while maintaining its small size. It has 3 variants, namely Reguler,LTS,MicroG.

GappsLite++ is made for those of you who like to change roms or android versions, you can use this for many devices and android versions in one flashable ZIP Read more
